# 图片工具
本工程基于Qt开发，其软件用于日常对图片的处理，避免麻烦的代码处理过程。

## 开发环境

windows 10 (64 bit)

QT Creator 4.12

Opencv-4.6.0（使用cmake-3.26.0编译）

QT 工程文件(.pro) 加入 opencv 路径。就达到使用 opencv 的目的

## 工程介绍

imageshow 是源代码文件

output 是编译结果，其中的 Image_show.exe 已经做了跨设备适配，下载到本地直接运行。
